jQuery.Kerning.js

This is a jquery plugin which adjust text kerning in HTML.

About

カーニング調整のためのjQueryプラグインです。

Example

あいうえお、かきくけこ。

あいうえお、かきくけこ。

HTML
<h1>あいうえお、かきくけこ。</h1>
<h1 class="kern">あいうえお、かきくけこ。</h1>
js 
$.getJSON("./kerning.json" , function(data) {
  $('h1.kern').kerning({"data":data});
});
CSS 
h1 {
  letter-spacing: 0.1em;
}
